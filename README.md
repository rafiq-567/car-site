# Car Showcase Website

A modern, responsive car showcase website built with Next.js and Tailwind CSS.

## Live Demo
Live Link: https://car-site-mocha.vercel.app/

## Github Repository: https://github.com/rafiq-567/car-site

## Features
- Browse luxury car collection
- Search by name or brand
- Filter by price range
- View full car details in a modal
- Contact form
- Fully responsive (mobile + desktop)

## Tech Stack
- Next.js
- Tailwind CSS
- Deployed on Vercel

## Getting Started
```bash
npm install
npm run dev
```
